Fix Visualization Errors

Error Message

Failed to load datasets index. Please run the update_index.bat script in the data/analysis folder and make sure the analyses_index.json file exists.

This error occurs because:

The visualization system is trying to find a file called analyses_index.json in a specific location, but it doesn't exist or can't be accessed.

Solution 1: Run the Fix Script

The easiest way to fix this is to run our fix_visualizations.bat script:

  1. Open a command prompt (CMD)
  2. Navigate to this directory (src/web)
  3. Run the fix script: 
    fix_visualizations.bat

This script will:

  • Scan your analysis folders
  • Generate the required index file in the correct location
  • Set up everything needed for the visualizations to work

Solution 2: Manual Steps

If the script doesn't work for any reason, you can perform these steps manually:

  1. Create these folders (if they don't exist): 
    data\web\visualizations\data
  2. Copy the update_index.bat script from data\analysis to data\web
  3. Run the script from the data\web directory

Solution 3: Use the Simple Visualization

We've created a simplified visualization that's more tolerant of different file structures:

Try accessing: 

http://localhost:8000/src/web/templates/simple_visualizations.html
instead of 
http://localhost:5000/visualizations.html

The simple visualization will try multiple paths to find your data files.
